### TensegritySim

This project uses the TensegritySim python package. It is added here as a git submodule in tensegrity-sim. To clone this repository with the submodule, use the following command:

```bash
git clone --recurse-submodules <this-repository-url>
```

If you have already cloned the repository without the submodule, you can add it using the following command:

```bash
git submodule update --init --recursive
```

### Dependencies
Reccomended to use a virtual environment to install the dependencies. To create a virtual environment, run the following command: `python3 -m venv .venv`. To activate the virtual environment, run the following command: `source .venv/bin/activate`

To install the dependencies, run the following command:

```bash
pip install -r requirements.txt
```
