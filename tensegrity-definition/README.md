A code repository that contains the code for the definition of a tensegrity structure, as well as other solvable solutions such as minimum mass or control solutions.

Some Rules so not to break the code:
1. Be sure to add any new methods to the src/Main folder.
2. Make sure the PYTHONPATH is set the location of the Tensegrity Definition folder.  i.e. If you cloned this repo into your documents folder, you would need to run: unset PYTHONPATH; export PYTHONPATH=~/Documents/TensegrityDefinition
3. Ensure that the cwd is also at the location of the Tensegrity Definition folder.
